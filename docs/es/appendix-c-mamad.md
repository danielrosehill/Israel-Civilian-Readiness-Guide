# Apéndice C: Inspección del Mamad (Espacio Protegido)

*Basado en las directrices del Comando del Frente Interior para espacios protegidos residenciales.*

### Inspección Estructural

| ELEMENTO | VERIFICACIÓN |
|---------|-------------|
| **Puerta Blindada** | ☐ **ABRE Y CIERRA FÁCILMENTE** · ☐ **MANIJA GIRA 90° HACIA ARRIBA** · ☐ **DOBLE CERROJO FUNCIONA** |
| **Sello de la Puerta** | ☐ **AISLAMIENTO DE GOMA PRESENTE** *No reseco ni agrietado — reemplace si está quebradizo* |
| **Prueba de Luz** | ☐ **SIN PENETRACIÓN DE LUZ AL SELLAR** *Luz = sello inadecuado* |
| **Ventana Exterior de Acero** | ☐ **ABRE Y CIERRA FÁCILMENTE** |
| **Ventana Interior de Vidrio** | ☐ **FUNCIONA CORRECTAMENTE** · *Doble hoja: retire ambas hojas durante escalada y guárdelas fuera de la habitación* |
| **Tuberías de Ventilación** | ☐ **AISLAMIENTO DE GOMA INTACTO** · ☐ **TORNILLOS DE LA CUBIERTA DE ACERO APRIETAN COMPLETAMENTE** |

### Contenido y Seguridad

| ELEMENTO | VERIFICACIÓN |
|---------|-------------|
| **Inflamable / Peligroso** | ☐ **NINGUNO** *Sin materiales inflamables, peligrosos o tóxicos* |
| **Vidrio / Cerámica** | ☐ **NINGUNO** *Sin vidrio, cerámica o porcelana que pueda romperse* |
| **Objetos Pesados** | ☐ **ESTANTES Y OBJETOS PESADOS FIJADOS A LAS PAREDES** |
| **Tanques de Gas** | ☐ **A 3+ METROS DE LAS PAREDES DEL CUARTO PROTEGIDO** |
| **Suministros de Emergencia** | ☐ **ALMACENADOS DENTRO O INMEDIATAMENTE CERCA** |
| **Ventilador de Techo** | ☐ **NINGUNO INSTALADO** *Prohibido en mamads* |
| **Ventilación** | ☐ **HABITACIÓN VENTILADA SI SE USA COMO DORMITORIO** · *NO debe usarse como cocina, baño o lavadero* |
