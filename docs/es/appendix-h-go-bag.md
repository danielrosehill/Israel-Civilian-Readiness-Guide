# Apéndice H: Recomendaciones para la Mochila de Emergencia

*Una lista detallada de empaque. Adapte según el tamaño y necesidades de su hogar.*

### Nivel 1 — Siempre Empacado (Crítico)

| | ELEMENTO | NOTAS |
|---|---------|-------|
| 📱 | Cargador de teléfono + cable | Uno por tipo de dispositivo |
| 📱 | Batería externa | 10,000+ mAh; verificar carga semanalmente |
| ⛑ | Medicamentos esenciales | Suministro para 7 días; rotar mensualmente |
| | Linterna + baterías de repuesto | O linterna de manivela |
| | Mascarillas N95 | Una por persona; selladas en bolsa |
| 💧 | Botella de agua | 1L mínimo por persona |
| | Silbato de emergencia | Audible desde bajo escombros |
| ⛑ | Kit de primeros auxilios | Vendas, antiséptico, analgésicos, tijeras |
| | Copias de esta guía | Plastificadas o en funda impermeable |

### Nivel 2 — Adiciones Recomendadas

| | ELEMENTO | NOTAS |
|---|---------|-------|
| 📻 | Radio AM/FM | A pilas o de manivela; escriba las frecuencias en ella |
| | Router de viaje + ethernet | Para refugio con internet por cable |
| | Adaptadores / regletas | Las tomas del refugio son compartidas |
| | Efectivo (NIS) | Billetes pequeños; los cajeros pueden estar fuera de servicio |
| | Pasaportes + DNI | Copias en bolsa impermeable |
| | Lista de contactos de emergencia | Impresa; los teléfonos se agotan |
| | Multiherramienta o navaja suiza | Cuchilla, abridor, destornillador |
| | Cinta adhesiva + bridas | Reparación universal |
| | Marcador permanente + papel | Para notas si no hay teléfono |

### Nivel 3 — Estancia Prolongada (4+ Horas / Nocturna)

| | ELEMENTO | NOTAS |
|---|---------|-------|
| 🍽 | Snacks no perecederos | Barras energéticas, frutos secos, frutas deshidratadas, galletas |
| 💧 | Agua adicional | 3L por persona por día |
| | Antifaz + tapones para oídos | Para dormir en refugio compartido |
| | Manta o saco de emergencia | Las mantas de emergencia de aluminio son compactas |
| | Muda de ropa | Incluyendo calcetines y ropa interior |
| | Bolsa de aseo | Cepillo de dientes, toallitas húmedas, gel desinfectante |
| | Entretenimiento | Libro, cartas, material para colorear para niños |
| 🎒 | Artículos religiosos | Sidur, tefilín (si aplica) |
| | Paraguas | Para caminar al refugio bajo lluvia |

!!! tip
    **Calendario de rotación:** Verifique medicamentos y batería externa mensualmente. Rote alimentos y agua cada 3 meses. Reemplace baterías cada 6 meses. Actualice documentos anualmente.
