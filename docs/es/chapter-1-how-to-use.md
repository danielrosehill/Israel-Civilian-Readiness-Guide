# Capítulo 1: Cómo Usar Este Documento

## 1.1 Preparación (Antes de Emergencias)

!!! tip
    **Imprima y prepare esta lista de verificación antes de necesitarla.** No espere a que suene una sirena para saber qué hacer.

Durante períodos de rutina:

- **Imprima varias copias** en papel resistente (o plastifique si es posible)
- **Adhiera una tira de cinta reflectante** a la portada o borde superior — visible al iluminar con una linterna en la oscuridad
- **Encuaderne o grape** las páginas juntas. Una carpeta de documentos o funda plástica funciona bien
- Prepare al menos **3 copias:**
  - En la nevera o junto a la puerta principal (la copia de trabajo)
  - En la mochila de emergencia
  - En el mamad / espacio protegido
- **Revise y reimprima** cada vez que se actualice la lista

## 1.2 Ejecución de la Lista de Verificación

**Método de dos personas (recomendado).** Una persona lee cada elemento en voz alta. La otra lo verifica físicamente y responde (por ejemplo, "VERIFICADO", "HECHO", "PENDIENTE"). Este es el método de desafío y respuesta utilizado en listas de verificación de aviación y militares.

| LISTA DE VERIFICACIÓN | CUÁNDO EJECUTAR |
|----------------------|-----------------|
| **Lista Maestra Completa** (Cap. 3) | Inicio de cualquier escalada o guerra · Después de una evacuación · Semanalmente durante conflicto sostenido |
| **Prueba Rápida — PAWS BED** (Cap. 2) | Diariamente en tiempos de guerra · Semanalmente durante tensión elevada · Mensualmente durante rutina |
| **Listas Situacionales** (Cap. 4) | Cada vez, durante tiempos de guerra o escalada |

!!! danger "Advertencia"
    Los elementos marcados con ⚠️ son **críticos** — no los omita bajo ninguna circunstancia.
