# Guía de Campo de Preparación para Tiempos de Guerra en Israel

**Guía de Campo y Listas de Verificación — V4**

*Por Daniel Rosehill + Claude Opus | 20 de marzo de 2026*

Una guía integral de preparación civil para tiempos de guerra en Israel, basada en las directrices del Comando del Frente Interior (Pikud HaOref). Esta guía proporciona listas de verificación imprimibles, procedimientos situacionales y orientación práctica para mantenerlo a usted y a su hogar seguros durante emergencias.

!!! tip "Listas de Verificación Interactivas"
    Las listas de verificación en esta guía usan casillas interactivas — márquelas a medida que revisa cada elemento. Para una copia física, descargue el PDF imprimible a continuación.
