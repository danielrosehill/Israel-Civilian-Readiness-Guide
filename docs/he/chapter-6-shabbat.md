# פרק 6: שבת / חג (יהודים שומרי מצוות)

*כשהטלפונים כבויים והצפירות הן ההתרעה היחידה. התכוננו לפני הדלקת נרות.*

## 6.1 לפני שבת / חג

| | פריט | בדיקה |
|---|------|-------|
| 📡 | **ערוץ 14 / גלי צה"ל** ⚠️ | ☐ **משדר בטלוויזיה לפני שבת** · ☐ **עוצמה נבדקה — נשמע מחדרי השינה** *ודאו שהטלוויזיה לא מושתקת* |
| 📻 | **רדיו חירום** ⚠️ | ☐ **תדר ופעולה מאומתים** · ☐ **עוצמה מקסימלית** · ☐ **מקור חשמל — חשמל או סוללות** |
| 🎒 | **תיק חירום** | ☐ **ליד הדלת** · ☐ **תכולה מאומתת** |
| 🎒 | **ציוד שבת** | ☐ **ארוז** *סידור, כוס קידוש, חטיפים למקלט* |
| 👟 | **נעליים** ⚠️ | ☐ **סגורות ליד המיטה וליד הדלת** *אל תסתמכו על כפכפים* |
| 👕 | **בגדים** | ☐ **מונחים ליד המיטה** *סט מלא, לא רק פיג׳מה* |
| 🔑 | **מפתחות** | ☐ **ליד דלת הכניסה** |
| | **פנס** | ☐ **בהישג יד** *מוכן מראש על השידה* |

!!! tip
    **פיקוח נפש** דוחה את כל איסורי השבת. אם נשמעת צפירה, מותר לכם — ו**חובה** — להשתמש בטלפון, לשאת חפצים ולעשות כל מה שנדרש כדי להגיע למקלט בבטחה. זו אינה קולא; זו **הלכה**.

## 6.2 במהלך שבת / חג

- **צפירות הן ההתרעה העיקרית שלכם.** השאירו חלונות מעט פתוחים כדי לשמוע צפירות חיצוניות
- **טלוויזיה או רדיו חייבים להישאר דלוקים ונשמעים** לאורך כל השבת — זהו מקור המידע היחיד שלכם בזמן אמת
- **הכירו את מסלול המקלט בחושך** — אין פנס טלפון זמין (אלא אם נשמעת צפירה)
- לאחר התרעה, **הישארו במקלט 10 דקות או עד ל"ניתן לצאת" בטלוויזיה/רדיו**

!!! danger "אזהרה"
    **אם שומעים צפירה בשבת:** עברו למקלט מיד. נשאו ילדים, טלפון, תיק חירום — כל מה שצריך. אין כאן שאלה הלכתית.

    *מבוסס על נוהל פיקוד העורף נכון ל-20 במרץ 2026. התייעצו עם הרב שלכם להנחיה הלכתית ספציפית.*
