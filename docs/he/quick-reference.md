# עיון מהיר

*קרעו או צלמו דף זה. תלו על המקרר, בממ״ד ובתיק חירום.*

## מצב יום בבית

- [ ] **טלפון — עליך, טעון, אפליקציית פיקוד העורף**
- [ ] **לבוש — בגדים מלאים, נעליים סגורות**
- [ ] **מפתחות — עליך או בדלת / ליד הדלת**
- [ ] **תיק חירום — ליד הדלת, סגור**
- [ ] **יציאה — מסדרון פנוי, דלת נפתחת מהר**
- [ ] **התרעת דפדפן — פעילה (אם ליד המחשב)**

## מצב לפני השינה

- [ ] **טלפון — בטעינה, לא במצב טיסה**
- [ ] **בגדים — מונחים ליד המיטה**
- [ ] **נעליים — סגורות ליד המיטה**
- [ ] **פנס — שידה, בהישג יד**
- [ ] **משקפיים — ליד המיטה**
- [ ] **מפתחות — ליד המיטה או ליד הדלת**
- [ ] **תיק חירום — ליד הדלת, מטען נייד בטעינה**
- [ ] **יציאה — מסדרון פנוי, לא נעול בנעילה כפולה**
- [ ] **שמיעה — ללא אטמי אוזניים**
- [ ] **תלויים — כולם ידועים**

## ⚠️ התרעת לילה — 30 השניות הראשונות

1. **רגליים על הרצפה ← לתוך הנעליים** (אותו מקום כל לילה)
2. **תפסו משקפיים + טלפון** מהשידה
3. **כדור קפאין *(אופציונלי)*** — לגימת מים, אם שומרים ליד המיטה
4. **לבשו בגדים** שהונחו ליד המיטה
5. **פנס** אם אין חשמל
6. **עברו למרחב המוגן** או לדלת הכניסה
7. **הורים:** אחד ← ילדים, השני ← תיק חירום (מוקצה מראש)

### המדריכים שלכם

**Herman החמור — מומחה לבריאות ובטיחות**
מופיע לצד רשימות ביקורת בטיחות, נוהלי מקלט ופריטים קריטיים.

**Corn העצלן — מומחה לרווחה**
מופיע לצד טיפול עצמי, רווחה והנחיות בריאות הנפש.
