# נספח ג: בדיקת ממ״ד (מרחב מוגן)

*מבוסס על הנחיות פיקוד העורף למרחבים מוגנים דירתיים.*

### בדיקה מבנית

| פריט | בדיקה |
|------|-------|
| **דלת פלדה** | ☐ **נפתחת ונסגרת בקלות** · ☐ **ידית מסתובבת 90° כלפי מעלה** · ☐ **נעילה כפולה פועלת** |
| **איטום דלת** | ☐ **בידוד גומי קיים** *לא יבש או סדוק — החליפו אם שביר* |
| **בדיקת אור** | ☐ **אין חדירת אור כשאטום** *אור = איטום לא מספק* |
| **חלון פלדה חיצוני** | ☐ **נפתח ונסגר בקלות** |
| **חלון זכוכית פנימי** | ☐ **פועל כראוי** · *כנפיים כפולות: הסירו את שתי הכנפיים בהסלמה ואחסנו מחוץ לחדר* |
| **צינורות אוורור** | ☐ **בידוד גומי שלם** · ☐ **ברגי כיסוי פלדה מתהדקים עד הסוף** |

### תכולה ובטיחות

| פריט | בדיקה |
|------|-------|
| **דליקים / מסוכנים** | ☐ **אין** *אין חומרים דליקים, מסוכנים או רעילים* |
| **זכוכית / קרמיקה** | ☐ **אין** *אין זכוכית, קרמיקה, חרסינה שעלולים להתנפץ* |
| **פריטים כבדים** | ☐ **מדפים ופריטים כבדים מקובעים לקירות** |
| **מיכלי גז** | ☐ **3+ מטר מקירות המרחב המוגן** |
| **ציוד חירום** | ☐ **מאוחסן בפנים או בסמוך** |
| **מאוורר תקרה** | ☐ **לא מותקן** *אסור בממ״דים* |
| **אוורור** | ☐ **החדר מאוורר אם משמש כחדר שינה** · *אסור להשתמש כמטבח, חדר אמבטיה או חדר כביסה* |
