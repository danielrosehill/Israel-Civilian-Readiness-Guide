# Annexe H : Recommandations pour le sac d'urgence

*Liste detaillee du contenu. Adaptez-la a la taille et aux besoins de votre foyer.*

### Niveau 1 — Toujours pret (Critique)

| | ELEMENT | NOTES |
|---|------|-------|
| 📱 | Chargeur de telephone + cable | Un par type d'appareil |
| 📱 | Batterie externe | 10 000+ mAh ; verifier la charge chaque semaine |
| ⛑ | Medicaments essentiels | Reserve de 7 jours ; renouveler mensuellement |
| | Lampe torche + piles de rechange | Ou lampe torche a manivelle |
| | Masques N95 | Un par personne ; scelles dans un sac |
| 💧 | Bouteille d'eau | 1L minimum par personne |
| | Sifflet d'urgence | Audible sous les decombres |
| ⛑ | Trousse de premiers secours | Bandages, antiseptique, antidouleur, ciseaux |
| | Copies de ce guide | Plastifiees ou dans une pochette etanche |

### Niveau 2 — Ajouts recommandes

| | ELEMENT | NOTES |
|---|------|-------|
| 📻 | Radio AM/FM | A piles ou a manivelle ; inscrivez les frequences dessus |
| | Routeur de voyage + ethernet | Pour un abri avec internet filaire |
| | Adaptateurs / multiprises | Les prises de l'abri sont partagees |
| | Especes (NIS) | Petites coupures ; les distributeurs peuvent etre hors service |
| | Passeports + cartes d'identite | Copies dans une pochette etanche |
| | Liste de contacts d'urgence | Imprimee ; les telephones peuvent tomber en panne |
| | Couteau suisse ou multi-outil | Lame, ouvre-boite, tournevis |
| | Ruban adhesif + colliers de serrage | Reparation universelle |
| | Marqueur permanent + papier | Pour des notes si pas de telephone |

### Niveau 3 — Sejour prolonge (4+ heures / nuit)

| | ELEMENT | NOTES |
|---|------|-------|
| 🍽 | En-cas non perissables | Barres energetiques, noix, fruits secs, crackers |
| 💧 | Eau supplementaire | 3L par personne par jour |
| | Masque de sommeil + bouchons d'oreilles | Pour dormir dans un abri collectif |
| | Couverture ou couverture de survie | Les couvertures de survie en aluminium sont compactes |
| | Vetements de rechange | Y compris chaussettes et sous-vetements |
| | Trousse de toilette | Brosse a dents, lingettes, gel hydroalcoolique |
| | Divertissement | Livre, jeux de cartes, coloriage pour les enfants |
| 🎒 | Objets religieux | Siddour, tefiline (le cas echeant) |
| | Parapluie | Pour aller a l'abri sous la pluie |

!!! tip
    **Calendrier de rotation :** Verifiez les medicaments et la batterie externe mensuellement. Renouvelez la nourriture et l'eau tous les 3 mois. Remplacez les piles tous les 6 mois. Mettez a jour les documents annuellement.
