# Avertissement

Ce document **n'est pas une ressource gouvernementale officielle**. C'est un guide pratique communautaire compile par Daniel Rosehill avec l'aide de l'IA (Claude Opus), base sur les directives publiquement disponibles du Commandement du Front Interieur israelien (Pikud HaOref / פיקוד העורף).

**Source de reference :** Recoupement avec les directives officielles du HFC sur oref.org.il, archivees le **20 mars 2026**. Les fichiers HTML sources sont conserves dans le repertoire `sources/hfc-official/`.

**Referez-vous toujours aux instructions officielles** du HFC, de Tsahal, de la police israelienne et des autorites locales. Ce document est fourni tel quel, sans garantie.
