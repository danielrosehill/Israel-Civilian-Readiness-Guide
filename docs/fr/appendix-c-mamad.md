# Annexe C : Inspection du Mamad (Espace protege)

*Base sur les directives du Commandement du Front Interieur pour les espaces proteges residentiels.*

### Inspection structurelle

| ELEMENT | VERIFICATION |
|------|-------|
| **Porte blindee** | ☐ **S'OUVRE ET SE FERME FACILEMENT** · ☐ **LA POIGNEE TOURNE A 90° VERS LE HAUT** · ☐ **LE DOUBLE VERROUILLAGE FONCTIONNE** |
| **Joint de porte** | ☐ **ISOLATION EN CAOUTCHOUC PRESENTE** *Non dessechee ni fissuree — remplacer si fragile* |
| **Test de lumiere** | ☐ **AUCUNE PENETRATION DE LUMIERE UNE FOIS SCELLE** *Lumiere = etancheite insuffisante* |
| **Fenetre exterieure en acier** | ☐ **S'OUVRE ET SE FERME FACILEMENT** |
| **Fenetre interieure en verre** | ☐ **FONCTIONNE CORRECTEMENT** · *Double battant : retirer les deux battants en cas d'escalade et les stocker hors de la piece* |
| **Tuyaux de ventilation** | ☐ **ISOLATION EN CAOUTCHOUC INTACTE** · ☐ **LES VIS DU COUVERCLE EN ACIER SE SERRENT COMPLETEMENT** |

### Contenu et securite

| ELEMENT | VERIFICATION |
|------|-------|
| **Inflammable / Dangereux** | ☐ **AUCUN** *Pas de materiaux inflammables, dangereux ou toxiques* |
| **Verre / Ceramique** | ☐ **AUCUN** *Pas de verre, ceramique, porcelaine susceptible de se briser* |
| **Objets lourds** | ☐ **ETAGERES ET OBJETS LOURDS FIXES AUX MURS** |
| **Bouteilles de gaz** | ☐ **A 3+ METRES DES MURS DE LA PIECE PROTEGEE** |
| **Fournitures d'urgence** | ☐ **STOCKEES A L'INTERIEUR OU A PROXIMITE IMMEDIATE** |
| **Ventilateur de plafond** | ☐ **AUCUN INSTALLE** *Interdit dans les mamads* |
| **Ventilation** | ☐ **LA PIECE EST VENTILEE SI UTILISEE COMME CHAMBRE** · *Ne doit PAS etre utilisee comme cuisine, salle de bain ou buanderie* |
