# Annexe F : Frequences radio d'urgence

*Si le telephone et Internet sont en panne, une radio AM/FM a piles ou a manivelle est votre bouee de sauvetage.*

| STATION | FM | AM | NOTES |
|---------|----|----|-------|
| **Reshet Bet** (כאן ב׳) | 95.5 / 97.8 | 657 / 1206 | Informations nationales principales ; annonces du HFC |
| **Galei Tzahal** (גלצ) | 102.3 / 103.0 | — | Radio de Tsahal ; mises a jour securitaires |
| **Reshet Aleph** (כאן א׳) | 100.3 / 104.0 | 531 / 1008 | Debats et informations nationales |
| **Reshet Gimel** (כאן ג׳) | 97.0 / 99.0 | — | Musique hebraique + bulletins d'information |
| **Kol Israel Arabic** | 99.5 | 576 / 711 | Alertes en arabe |
| **Reka** (כאן רק״ע) | 101.5 / 104.5 | 972 | Russe/amharique/francais/anglais |

!!! tip
    - Les frequences FM varient selon la region — balayez les frequences voisines si pas de signal
    - **L'AM est plus fiable** en cas d'urgence — plus longue portee, fonctionne mieux a l'interieur des batiments
    - En cas d'urgence, toutes les stations diffusent simultanement les annonces du HFC
    - **Inscrivez vos frequences locales sur la radio elle-meme** avec un marqueur permanent
