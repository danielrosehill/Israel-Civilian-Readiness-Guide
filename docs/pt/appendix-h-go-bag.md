# Apendice H: Recomendacoes para Mochila de Emergencia

*Uma lista detalhada de itens. Adapte ao tamanho e necessidades da sua familia.*

### Nivel 1 — Sempre Embalado (Critico)

| | ITEM | NOTAS |
|---|------|-------|
| 📱 | Carregador de telefone + cabo | Um por tipo de dispositivo |
| 📱 | Power bank | 10.000+ mAh; verifique a carga semanalmente |
| ⛑ | Medicamentos essenciais | Suprimento de 7 dias; faca rodizio mensalmente |
| | Lanterna + pilhas reserva | Ou lanterna de manivela |
| | Mascaras N95 | Uma por pessoa; lacrada em saco |
| 💧 | Garrafa de agua | 1L minimo por pessoa |
| | Apito de emergencia | Audivel debaixo de escombros |
| ⛑ | Kit de primeiros socorros | Ataduras, antisseptico, analgesico, tesoura |
| | Copias deste guia | Plastificadas ou em envelope impermeavel |

### Nivel 2 — Adicoes Recomendadas

| | ITEM | NOTAS |
|---|------|-------|
| 📻 | Radio AM/FM | Pilhas ou manivela; escreva as frequencias nele |
| | Roteador portatil + ethernet | Para abrigo com internet cabeada |
| | Adaptadores / extensoes | Tomadas de abrigo sao compartilhadas |
| | Dinheiro (NIS) | Notas pequenas; caixas eletronicos podem ficar fora do ar |
| | Passaportes + documentos de identidade | Copias em bolsa impermeavel |
| | Lista de contatos de emergencia | Impressa; telefones descarregam |
| | Canivete suico ou multi-ferramenta | Lamina, abridor, chave de fenda |
| | Fita adesiva + abracasticas | Reparo universal |
| | Caneta permanente + papel | Para anotacoes se o telefone falhar |

### Nivel 3 — Estadia Prolongada (4+ Horas / Pernoite)

| | ITEM | NOTAS |
|---|------|-------|
| 🍽 | Lanches nao pereciveis | Barras de energia, nozes, frutas secas, biscoitos |
| 💧 | Agua extra | 3L por pessoa por dia |
| | Mascara de dormir + tampoes de ouvido | Para dormir em abrigo compartilhado |
| | Cobertor ou manta de emergencia | Mantas de aluminio de emergencia sao compactas |
| | Muda de roupa | Incluindo meias e roupa intima |
| | Necessaire | Escova de dentes, lencos umedecidos, alcool em gel |
| | Entretenimento | Livro, cartas, material de colorir para criancas |
| 🎒 | Artigos religiosos | Siddur, tefilin (se aplicavel) |
| | Guarda-chuva | Para caminhar ate o abrigo na chuva |

!!! tip
    **Cronograma de rodizio:** Verifique medicamentos e power bank mensalmente. Faca rodizio de comida e agua a cada 3 meses. Substitua pilhas a cada 6 meses. Atualize documentos anualmente.
