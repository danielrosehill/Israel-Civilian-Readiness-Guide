# الفصل الثاني: اختبار سريع — "PAWS BED"

*فحص جاهزية دوري سريع. يمكن إتمامه في أقل من دقيقتين.*

| # | الفحص | التحقق |
|---|-------|--------|
| **P** | 📱 **الهاتف (Phone)** | ☐ **مُشغّل · مشحون · تطبيق HFC يعمل · المنطقة الصحيحة** |
| **A** | 📡 **التنبيهات (Alerts)** | ☐ **التنبيهات اللاسلكية مُفعّلة · تجاوز وضع عدم الإزعاج مُتحقق** |
| **W** | 💧 **الماء (Water)** | ☐ **مخزون 72 ساعة متاح · غير منتهي الصلاحية** |
| **S** | 🛡 **الملجأ (Shelter)** | ☐ **أقرب 3 ملاجئ معروفة · المسارات قابلة للمشي** |
| **B** | 🎒 **الحقيبة (Bag)** | ☐ **بجانب الباب · مغلقة · المحتويات مُتحقق منها** |
| **E** | 🚪 **المخرج (Exit)** | ☐ **الممر خالٍ · الباب يمكن فتحه بسرعة** |
| **D** | 👥 **المعالون (Dependents)** | ☐ **جميع أفراد الأسرة حاضرون أو تم الاتصال بهم** |

**وسيلة التذكر: PAWS BED** — *"تحقق من PAWS قبل BED" (تحقق من الأساسيات قبل النوم)*
