# الملحق ج: فحص المم"د (المساحة المحمية)

*استناداً إلى إرشادات قيادة الجبهة الداخلية للمساحات المحمية السكنية.*

### الفحص الهيكلي

| العنصر | التحقق |
|------|-------|
| **باب الانفجار** | ☐ **يُفتح ويُغلق بسهولة** · ☐ **المقبض يدور 90° للأعلى** · ☐ **القفل المزدوج يعمل** |
| **عازل الباب** | ☐ **العزل المطاطي موجود** *ليس جافاً أو متشققاً — استبدله إذا كان هشاً* |
| **اختبار الضوء** | ☐ **لا يتسرب ضوء عند الإغلاق** *الضوء = عزل غير كافٍ* |
| **النافذة الخارجية الفولاذية** | ☐ **تُفتح وتُغلق بسهولة** |
| **النافذة الداخلية الزجاجية** | ☐ **تعمل بشكل صحيح** · *ذات الجناحين: أزل كلا الجناحين أثناء التصعيد وخزّنهما خارج الغرفة* |
| **أنابيب التهوية** | ☐ **العزل المطاطي سليم** · ☐ **براغي الغطاء الفولاذي تُشدّ بالكامل** |

### المحتويات والسلامة

| العنصر | التحقق |
|------|-------|
| **مواد قابلة للاشتعال / خطرة** | ☐ **لا يوجد** *لا مواد قابلة للاشتعال أو خطرة أو سامة* |
| **زجاج / سيراميك** | ☐ **لا يوجد** *لا زجاج أو سيراميك أو بورسلين يمكن أن يتحطم* |
| **عناصر ثقيلة** | ☐ **الأرفف والعناصر الثقيلة مثبتة على الجدران** |
| **خزانات الغاز** | ☐ **على بعد 3+ أمتار من جدران الغرفة المحمية** |
| **مستلزمات الطوارئ** | ☐ **مخزنة بالداخل أو بالقرب مباشرة** |
| **مروحة سقف** | ☐ **غير مُثبّتة** *محظورة في المم"د* |
| **التهوية** | ☐ **الغرفة مُهوّأة إذا استُخدمت كغرفة نوم** · *يجب ألا تُستخدم كمطبخ أو حمام أو غرفة غسيل* |
