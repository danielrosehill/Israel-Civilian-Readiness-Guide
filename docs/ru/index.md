# Полевое руководство по готовности к военному времени в Израиле

**Полевое руководство и чек-листы — V4**

*Автор: Daniel Rosehill + Claude Opus | 20 марта 2026*

Комплексное руководство по гражданской готовности к военному времени в Израиле, основанное на рекомендациях Командования тыла (Pikud HaOref). Это руководство содержит чек-листы для печати, ситуационные процедуры и практические рекомендации для обеспечения безопасности вас и вашей семьи во время чрезвычайных ситуаций.
